8.CSS盒模型代码缩写/单位和值

发表于 2019-04-20 | 分类于 CSS

颜色值缩写
16进制的色彩值如果每两位的值相同，可以缩写一半
p{color:#336699;}
缩写为
p{color:#369;}

字体缩写
body{font-style:italic;
font-variant:small-caps;//小写
font-weight:bold;//粗体
font-size:12px;
line-height:1.5em;
font-family:”宋体”,sans-serif;//字体从左往右检索，如若全部检索失败使用默认字体 sans-serif-英文字体
//:后面汉字用””，字母不需
}
缩写为
body{font:tialic small-caps bod 12px/1.5em “宋体”,sans-serif;}//缩写为font:…,line用/隔开
至少指定font-size和font-family属性，其他未指定使用默认值 font-size和line-height中间加/
常用
body{font:12px/1.5em “宋体”,sans-serif;}

颜色值
三种方法表示
1.英文命令 p{color:red;}
2.RGB(红绿蓝) p{color:rgb(122,42,143);}//0-255整数
p{color:rgb(34%,23%,54%);}//百分比比例
3.十六进制 p{color:#00ffff;}//00-ff

长度
三种单位
1.像素（px）：与显示器的实际像素有关
2.em：相当于已定义字体的像素倍数//以父元素为标准
3.百分比 p{font-size:12px;line-height:130%/1.3em/15.6px}
