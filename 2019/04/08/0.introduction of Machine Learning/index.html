1.机器学习是什么?
Machine Learning≈Looking for a Function form Data 根据所给的数据，找出一个最适当的函数。 2.机器学习的三个步骤 (1)Define a set of function(函数集合):确定Model
(2)Goodness of function:通过Training Data找出最好的函数 (3)Pick the “Best” Function(f):Using f Testing

3.Learning Map

蓝色scenario：有什么类型的training data 红色task：监督学习中的输出是什么 绿色Method：模型方法

（1）Supervised Learning(监督学习): 给定一个function：对于一个input输入要输出对应确定的输出（label) 1）Regression(回归)：The output of the target function f is “scalar”标量 预测PM2.5/连续函数下进行预测

2）Classification(分类)： Binary Classification（二分类）：输出yes or no/垃圾邮件过滤 Multi-class Classification（多分类）：输出多个类型/新闻分类 Linear Model（线性模型）：处理简单问题 Non-Linear Model（非线性模型）：处理复杂问题 包括Deep Learning（深度学习）：Each possible object is a class 比如：Image Recognition：每个识别结果是一个class Playing GO:棋盘每一个落子位置是一个class 3）Structured Learning（结构化学习）：Beyond Classification ,输出的是有结构的内容 （2）Semi-supervised Learning（半监督学习）：Hard to collect a large amount of labelled data，数据不足时用来学习，利用已有的Labelled和Unlabeled data （3）Transfer Learning（迁移学习）：已有一部分Labelled data和一部分Data（not related to the task considered） （4）Unsupervised Learning（无监督学习）：只有Labelled data 机器阅读：让机器阅读大量的文章，自己理解其中的意思 （5）Reinforcement Learning（强化学习）：监督学习是Learning for teacher 强化学习是Learning for critics，只知道自己输出的结果好还是不好，自己反复 来修正结果
