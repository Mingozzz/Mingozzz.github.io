html中标签元素类型：块状元素、内联元素(行内元素)、内联块状元素。

…

内联元素:

块状元素
a{display:block;} /将内联元素a转换为块状元素/
块状元素独占一行 高度、宽度、行高、顶和底边距都可设置

内联元素
div{display:inline;} /将块状元素div转换为内联元素/

内联块状元素
a{display:inline-block;} /将内联元素a转换为内联块状元素/

盒模型
div{border 2px solid/dotted/dashed red;} /边框的粗细、样式（实线、点线、虚线）、颜色/
div1{border-bottom/top/right/left:1px dashed red;} /单独边框设置/

padding:20px; /填充/

margin:10px; /边界/
}
