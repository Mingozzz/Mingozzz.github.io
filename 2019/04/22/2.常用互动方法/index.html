1.输出内容document.write()

  1. 直接输出
    document.write("Hello,world!")//用""括起

  2. 变量输出
    var mystr="Hello,world!"; document.write(var);//直接写变量名

  3. 多项内容用+连接
    var mystr="Hello,world!"; document.write(var+"good");//直接写变量名

  4. 输出HTML标签
    document.write("Hello,<br>world!")//用""括起

    var mystr="Hello,world!"; document.write(var+"<br>")
